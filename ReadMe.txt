To use this on localhost follow the steps below.

Make a virtual environment.

	mkvirtualenv Project_flasksite

	setprojectdir . 
	
	pip install flask
	
	set FLASK_APP=<app.py> // pick main file 

	flask run // this will start the localhost webpage

	deactivate // to stop working on project

	workon Project_flasksite // type this in the terminal to activate the project
